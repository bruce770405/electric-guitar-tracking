## 🎵 必備音階（依進度解鎖）

### 小調五聲音階 第 1 把位（Am Pentatonic, 第 5 格起）
```
e|---------------5--8--
B|---------5--8--------
G|------5--7-----------
D|---5--7-------------
A|5--7----------------
E|5--8----------------
```
根音：A（第 6 弦第 5 格）。這是 solo 的萬用起手式。

### 爬格子熱身 (Chromatic 1-2-3-4)
```
e|-1-2-3-4-----------
B|---------1-2-3-4---
... 逐弦進行，下上交替撥弦，節拍器慢速乾淨
```

## 🎸 常用 Power Chord 型

```
根音在第6弦:        根音在第5弦:
E|---            E|---
B|---            B|---
G|---            G|---
D|-7-            D|---
A|-7-            A|-7-
E|-5-  (A5)      E|-5-? → A弦根音型
```
口訣：食指壓根音，無名指 + 小指壓低兩條弦高 2 格。

## 🛠️ 推薦工具

- **節拍器**：手機 App「Soundbrenner」或「Pro Metronome」
- **調音器**：「GuitarTuna」
- **抓歌/變速**：「Moises」或 YouTube 0.5x 速度
- **Backing Track**：YouTube 搜「Am backing track」「Blues backing track in A」
- **錄音**：手機備忘錄即可，重點是「能回放聽自己」

## 📌 練習心法備忘

1. **慢而乾淨 > 快而髒**：先在能彈乾淨的最慢速度站穩
2. **節拍器是親媽**：每個技巧都跟節拍器，每次 +5 BPM
3. **錄音不騙人**：自己彈時感覺良好，錄起來才聽得出問題
4. **天天碰琴 > 假日狂練**：10 分鐘也好，保持手感最重要
5. **單點突破**：一次只專注修一個問題，別貪多
