# 🎸 電吉他半年進階計畫 (6-Month Electric Guitar Plan)

> 從「生鏽回鍋」到「能即興、能產出作品」的 24 週系統化練習

---

## 👤 學員背景

| 項目 | 內容 |
|---|---|
| 程度 | 曾有約半年木/電吉他底子，闊別 10+ 年，目前重拾第 1 週 |
| 定位 | **生鏽的回鍋初學者** — 有肌肉記憶殘留，需「喚醒 + 修正壞習慣」，比全新手快但不可直接跳進階 |
| 每週時間 | 輕量 3–5 小時／週（每天約 30 分鐘，週末可多） |
| 發展方向 | **全面均衡** — 技巧 / 樂理 / 律動 / 各曲風（Rock・Blues・Funk・Pop）都碰 |
| 追蹤形式 | Markdown 計畫總覽 + 每週日誌 + HTML 可視化 + 任務制打勾 |

## 🎯 半年總目標（畢業標準）

完成後你應該能夠：

- [ ] 在不看手的狀態下，乾淨彈出五聲音階 5 個把位（全指板）
- [ ] 推弦音準穩定（全音 / 半音可聽辨），Vibrato 有控制力
- [ ] 交替撥弦穩定 BPM ≥ 120（16 分音符）
- [ ] 看到和弦進行能即興 solo（至少 Blues 12 小節 / 一個流行曲）
- [ ] 理解 CAGED 系統與大調/小調音階級數，能自行抓歌
- [ ] 完整彈奏並**錄音產出 3 首作品**（不同曲風各一）

## 🗺️ 階段地圖 (6 Phases / 24 Weeks)

### Phase 1 ｜ M1 喚醒與校正 (W1–W4)
**目標**：找回手感、建立正確姿勢與 Picking、節拍器內化
- 左手：按弦力道、指尖立起、不悶到鄰弦
- 右手：Pick 握法、下上交替撥弦、撥弦角度
- 核心：每日節拍器熱身、爬格子 (1-2-3-4)、單音計時器
- 樂理：音名、指板前 5 格、TAB / 六線譜重新熟悉

### Phase 2 ｜ M2 基礎重建 (W5–W8)
**目標**：五聲音階 + Power Chord + 悶音 + 基礎推弦
- 小調五聲音階第 1 把位（Am pentatonic）
- Power Chord 全指板移動 + Palm Muting 悶音律動
- 基礎推弦（半音/全音）入門
- 樂理：音程概念、根音定位

### Phase 3 ｜ M3 技巧擴張 (W9–W12)
**目標**：搥勾滑 (HOPO)、推弦音準、提速、節奏切分
- 搥弦 Hammer-on / 勾弦 Pull-off / 滑音 Slide
- 推弦音準校正（對照空弦）+ Vibrato 顫音
- 交替撥弦提速（節拍器階梯式加速）
- 樂理：切分音、附點、休止符律動

### Phase 4 ｜ M4 樂理與指板 (W13–W16)
**目標**：打通指板、CAGED、調式級數
- CAGED 系統五個把位串聯
- 大調 / 小調音階全指板
- 和弦級數 (I–IV–V、ii–V–I) 與抓歌
- 樂理：順階和弦、五度圈入門

### Phase 5 ｜ M5 曲風探索 (W17–W20)
**目標**：四大曲風各取一段 riff + solo
- **Rock**：經典 riff + 破音 tone、點弦初探
- **Blues**：12 小節進行 + Blues scale + 即興問答句
- **Funk**：16 分音符切分刷弦、悶音 cutting、律動
- **Pop**：clean tone 分解和弦、情感 solo

### Phase 6 ｜ M6 進階整合 (W21–W24)
**目標**：即興 + 完整曲目 + 錄音產出
- 即興 solo（套用音階 + 樂句庫）
- 挑選 3 首歌完整練成（不同曲風）
- 錄音 / 簡單後製，產出作品
- 自我複盤 + 下半年方向規劃

## ⏱️ 每日 30 分鐘練習結構（黃金比例）

| 區塊 | 時間 | 內容 |
|---|---|---|
| 🔥 熱身 | 5 min | 爬格子 + 節拍器（慢速乾淨優先） |
| 🎯 技巧 | 10 min | 當週主題技巧（音階/推弦/撥弦…） |
| 🎵 應用 | 10 min | 當週練習曲 / riff / 樂句 |
| 🎼 樂理/聽力 | 5 min | 當週樂理點 or 抓歌 / 聽歌打拍 |

> 週末可把「應用」與「樂理」區塊加長，或加入錄音檢視。

## 📐 自我檢測標準（每月底）

每個月底用手機錄一段，對照以下指標（1–5 分自評）：
1. **乾淨度**：有無雜音、悶音是否到位
2. **節奏**：是否跟得上節拍器、切分是否準
3. **音準**：推弦是否到位、按弦是否飽滿
4. **流暢度**：換把位 / 換和弦是否卡頓
5. **音樂性**：有沒有「好聽」、強弱動態

> 規則：**慢而乾淨 > 快而髒**。任何技巧先在「能彈乾淨的最慢速度」站穩，再用節拍器每次 +5 BPM。

## 🚀 如何開始

1. 開啟 `plan.html` 用瀏覽器閱讀完整計畫（可勾選追蹤進度）
2. 每週開始前，複製 `logs/_template.md` 成 `logs/week-XX.md`
3. 每天練完打勾、記下 BPM 與心得
4. 月底錄音自評，回填檢測分數
5. 在這個對話裡隨時找我調整課表、解答卡關、推薦曲目 💪
